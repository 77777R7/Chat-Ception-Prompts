# Contributing

Contributions are welcome.

Good prompt changes should:

- Keep the visible UI text in English.
- Preserve the recursive screenshot structure.
- Improve realism, readability, or generation reliability.
- Avoid adding watermarks, random captions, poster layouts, or unrelated UI.
- Avoid presenting generated screenshots as real private conversations.

## Suggested Pull Request Format

- What prompt changed
- Why the change improves generation quality
- Any known tradeoffs
- Example output, if available
